## ⚖️ Licencia (Update: MIT → GPLv3)

Este proyecto ha pasado de una licencia MIT a **GNU GPLv3**. 

### ¿Por qué el cambio?
He decidido migrar a una licencia **Copyleft** para asegurar que estos juegos en Pascal sigan siendo libres para siempre. 
* **Si usas este código:** Puedes hacerlo libremente para aprender o jugar o para cualquier otro proposito.
* **Si mejoras el código:** Eres libre de distribuirlo, pero **debes** compartir tus mejoras bajo esta misma licencia.
* **Transparencia:** Al aparecer en motores de búsqueda como DuckDuckGo, quiero que cualquier persona que encuentre este repositorio sepa que la libertad del código está protegida.

Consulta el archivo [LICENSE](LICENSE) para leer el texto completo de la licencia.
